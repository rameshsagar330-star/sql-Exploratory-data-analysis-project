# sql-Exploratory-data-analysis-project
## Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** using SQL to uncover meaningful patterns, trends, and anomalies within a raw dataset. The primary goal was to transform unstructured data into actionable business insights by applying advanced querying techniques.
## Key Objectives
  **1.Data profiling:** Assessing the quality, completeness, and distribution of the data.
  **2.Trend Analysis:**  Assessing the quality, completeness, and distribution of the data.
  **Customer/Product Insights:** Segmenting data to find top-performing categories and high-value behaviors.
  **Data Cleaning:** Handling null values, duplicates, and inconsistent formatting directly within SQL.
## Technical Skills Applied
  **Window Functions:** Implementing RANK(), DENSE_RANK(), and ROW_NUMBER() for competitive analysis.
  **CTE’s & Subqueries:** Organizing complex logic into readable, modular code.
  **Joins:** Merging multiple relational tables to create a unified view for analysis.
  **Date/Time Manipulation:** Extracting year, month, and day trends for temporal analysis.
